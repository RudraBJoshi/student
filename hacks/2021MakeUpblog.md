# MakeUp of 2021 Practice Exam MCQ — Reflection

**Score: 66/70**

Overall, I'm satisfied with this result. Getting 66 out of 70 on the makeup practice MCQ shows a strong grasp of most AP CSP concepts, but there are clear gaps worth addressing before the real exam.

---

## Strengths

### Networking and the Internet
Questions covering how the internet works, the relationship between the internet and World Wide Web, bandwidth effects, and how data is transmitted on the internet were all answered correctly. I clearly have a solid conceptual foundation here.

### Security and Cryptography
I correctly answered questions on multifactor authentication, public key encryption, symmetric encryption, and weak passwords. Security concepts are a consistent strength.

### Data and Binary Representation
Questions on binary numbers, decimal representation, digital signals, and adding binary zeros all went well. I understand how data is stored and manipulated at a low level.

### Algorithms and Efficiency
Most algorithmic thinking questions — parallel computing, driving route algorithms, binary search efficiency, and crowdsourcing — were answered correctly. I can reason about algorithm behavior and tradeoffs.

### Creative Commons and Ethics
Questions on Creative Commons licensing and digital divide were handled well, showing awareness of the social and ethical dimensions of computing.

### Simulation and Modeling
I got most simulation questions right, showing I understand when and why simulations are used in computing.

---

## Weaknesses

### List and Data Structure Operations
I struggled with questions involving list manipulation — specifically moving elements from the end of a list to the beginning. This points to a gap in understanding how list traversal and index-based operations work step by step.

### Loop Logic and Infinite Loops
The question about initial values that cause an infinite loop tripped me up. I need to practice tracing through loop conditions carefully, especially edge cases where the condition never becomes false.

### Game Logic / Procedural Simulation
The question about moving a piece on a game board was missed. These questions require careful step-by-step simulation of code, and I need to slow down and trace through each instruction precisely.

### Code Debugging (Procedures)
Missing the spinner behavior or procedure error questions suggests I sometimes misread what a segment of code actually does versus what I expect it to do. Careful code tracing is something to practice more.

---

## Plan Going Forward

1. **Practice list operations** — trace through append, remove, insert, and index-based operations by hand.
2. **Loop tracing** — for any loop question, explicitly write out the value of each variable at each iteration.
3. **Code simulation** — slow down on questions with code blocks. Read each line as the computer would, not as I want it to behave.
4. **Review missed questions** — go back to each of the 4 wrong answers, understand why my answer was wrong, and understand why the correct answer is right.

Overall this was a strong performance, but consistent mastery requires closing these smaller gaps in code tracing and list logic.
